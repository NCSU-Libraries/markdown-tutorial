We'll now learn how to make links to other web sites on the world wide web.

To create a link, you wrap the link text that you want to appear for the link in brackets ( [ ] ), and then you wrap the link in parenthesis ( ( ) ). For example, to create a hyperlink to www.github.com, with a link text that says, Visit GitHub!, you'd write this in Markdown:

[Visit GitHub!](www.github.com)

In the box below, make a link to www.google.com, with link text that says "Search for it."
