We'll now learn how to make links to other web sites on the world wide web.

To create a link, you wrap the link text that you want to appear for the link in brackets ( [ ] ), and then you wrap the link in parenthesis ( ( ) ). For example, to create a hyperlink to www.github.com, with a link text that says, Visit GitHub!, you'd write this in Markdown:

[Visit GitHub!](www.github.com)

You can make headings links too!

In addition to traditional links, markdown can also format reference links. As the name implies, the link is actually a reference to another place in the document. This is a great way to use the same link in multiple places in a document Here's an example of what we mean:

Here's [a link to something][something]. Here's [a link to something else][something-else]. But here is [another link to something][something]. [something]: www.something.com [something-else]: www.somethingelse.com

As you can see in the example above we used the 'something' link twice, but only had to enter the url for the link at the bottom of the page, like a reference or citation that you are used to using in other types of writing.

Reference links don't appear in the rendered Markdown. You define them by providing the same tag name wrapped in brackets, followed by a colon, followed by the link.
