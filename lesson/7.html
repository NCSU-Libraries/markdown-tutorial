Markdown has several ways of formatting paragraphs.

Let's take a few lines of poetry as an example. Suppose you want to write text that looks like this:

Do I contradict myself?
Very well then I contradict myself,
(I am large, I contain multitudes.)

Now, you might think that simply typing each verse onto its own line would be enough to solve the problem:

Do I contradict myself? Very well then I contradict myself, (I am large, I contain multitudes.)

Unfortunately, you'd be wrong! This Markdown would render simply as a single straight line.

If you forcefully insert a new line by pressing the return key at the end of each line, markdown will render each like as a separate paragraph, resulting in an extra blank line between each line of text

This is what's known as a hard break; what our poetry asks for is a soft break. You can accomplish this by inserting two spaces at the end of each line. This is not possible to see, since spaces are invisible, but it'd look something like this(the dots represent spaces):

Do I contradict myself?路路 Very well then I contradict myself,路路 (I am large, I contain multitudes.)
